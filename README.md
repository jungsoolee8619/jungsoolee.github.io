# Portfolio Website

A clean and professional portfolio website template based on Bootstrap 4, similar to the design at leebebeto.github.io.

## Features

- ✨ Minimalist, clean design
- 📱 Fully responsive on mobile and desktop
- 🎓 Academic-style layout perfect for researchers and professionals
- 🖼️ Project/publication showcase with images
- 🔗 Easy integration with social profiles
- 📄 Modern typography and spacing

## Setup

1. **Enable GitHub Pages**:
   - Go to repository Settings → Pages
   - Set source to "Deploy from a branch"
   - Select your default branch (main/master) and root folder

2. **Customize Content**:
   - Edit `index.html` with your information
   - Replace placeholder text with your details
   - Update social media links

3. **Add Images**:
   - Create an `images/` folder if it doesn't exist
   - Add your profile picture as `images/profile.jpg`
   - Add publication thumbnails as `images/publication1.png`, etc.
